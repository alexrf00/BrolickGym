## Getting Started
First, run dependencies installation package:
```bash
npm install
```

## Second setup infisical CLI installation
```bash
npm install -g @infisical/cli
# and navigate to project root and run
npx infisical login
# and select project
npx infisical init
```

## Third run project
```bash
npm run app
```
